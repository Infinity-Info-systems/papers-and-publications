# Publication Registry

## Purpose

Use this registry to track papers, abstracts, citation files, and publication-ready assets.

## Publication Sections

### 1. SSRN

- working papers
- abstracts
- citation links
- revision history

### 2. Conference Submissions

- submission drafts
- acceptance status
- presentation materials
- evidence of submission

### 3. Bibliography

- master reference list
- citation formatting
- source grouping
- repository cross-links

### 4. Research Abstracts

- executive summary abstract
- methodology abstract
- impact abstract
- topic abstract

### 5. Citation Assets

- BibTeX entries
- DOI references
- author metadata
- reuse notes

## Recommended Actions

- keep one master citation source
- link each publication to the relevant repo
- maintain a submission log
- separate published, draft, and planned items clearly
